# Домашня робота №5: Основи CSS та Git

Це навчальний проект, створений під час проходження курсу Full Stack Developer. 

## 📝 Опис проекту
Проект демонструє навички роботи з селекторами CSS, розуміння специфічності (ваги) селекторів та базові навички керування версіями за допомогою Git.

## 🚀 Функціональність та особливості
* Робота з CSS Specifity: У файлі представлені різні комбінації селекторів (ID, класи, теги) для демонстрації пріоритетів стилів.
* Скидання стилів: Використано універсальний селектор для обнулення стандартних відступів браузера.
* Центрування елементів: Реалізовано вирівнювання заголовків та контенту.
* GitHub Pages: Проект опубліковано як статичну веб-сторінку.

## 🛠 Технології
* HTML5
* CSS3
* Git / GitHub
